# Agents Directory

This folder will contain custom Cursor agent configurations for the AI Media Buyer project.

- Define specialized agents here to work with the Google Ads Campaign Intelligence OS.
- Keep each agent focused (e.g., "report-ingestion-agent", "campaign-strategy-agent", "frontend-ux-agent").
- Align all agents with the rules in `../rules/ai-media-buyer-architecture.mdc`.

You can add more detailed agent files in this directory as needed.
